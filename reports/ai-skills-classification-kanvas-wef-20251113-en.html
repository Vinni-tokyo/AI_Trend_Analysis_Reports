1. KANVAS: Explainable AI-Based GenAI Job Classification

Current Status: Published in September 2025 in MDPI's Algorithms journal, KANVAS (Kolmogorov-Arnold Network Versatile Algorithmic Solution) represents a groundbreaking framework for distinguishing traditional AI roles from emerging GenAI-related positions. Analyzing 9,357 online job advertisements from EU countries between January 2023 and May 2025, the research revealed that GenAI positions prioritize prompt engineering and LLM integration, while traditional roles emphasize statistical modeling and legacy toolkits.

‚Ä¢ Open Source: Implementation code publicly available on GitHub (accessed August 2025)

‚Ä¢ GenAI Positions: Prompt engineering, LLM integration, generative AI tool utilization required

‚Ä¢ Practical Insights: Applicable to curriculum design, reskilling programs, workforce policy development

4. LinkedIn Skills Graph: Interconnection Mapping of 39,000 Skills

Current Status: LinkedIn's skills taxonomy has grown 35% since February 2021, comprising approximately 39,000 skills, 374,000 aliases, and over 200,000 edges (skill connections) across 26 locales. Each skill is represented as a "node" within the taxonomy, forming a hierarchical skill network through "edges" called knowledge lineages. This dynamic skills graph continuously learns from the behavior of over 1 billion LinkedIn users, automatically detecting and classifying emerging skills.

Key Terminology

Kolmogorov-Arnold Networks (KAN) - Interpretable neural networks based on B-splines. Unlike traditional MLPs, KAN places learnable activation functions on edges rather than nodes, improving model transparency
Explainable AI (XAI) - Techniques that explain AI model prediction rationale in human-understandable forms. Methods like SHAP (Shapley Additive Explanations) ensure transparency of black-box AI
Skills Ontology - Knowledge graph structuring relationships between skills (prerequisites, dependencies, similarities). While taxonomies define only hierarchical structures, ontologies express complex interrelationships
GenAI Skills - Skillset specialized for generative AI. Distinct capability group from traditional AI skills (machine learning, statistical modeling), including prompt engineering, LLM integration, generative AI tool utilization, model fine-tuning
AI Substitution Capacity Assessment - Metric evaluating how replaceable specific skills are by AI. WEF framework uses 5-level assessment from three perspectives: theoretical knowledge provision, problem-solving, physical presence
Skills Intelligence - Process using AI and machine learning to collect, analyze workforce data and translate into actionable insights. Enables governments and enterprises to dynamically define, measure, and adapt workforce needs in real-time
Linked Open Data - Approach interconnecting taxonomies like ESCO, O*NET, Lightcast to build global skills knowledge graph. Each framework functions as foundational node, achieving international interoperability
SHAP (Shapley Additive Explanations) - Explainable AI method based on game theory's Shapley values. Quantitatively evaluates how much each feature in a machine learning model contributed to prediction results
