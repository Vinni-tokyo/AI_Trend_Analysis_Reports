플랫폼 상세 분석

1. Workday Skills Cloud: 20만+ 스킬의 유니버설 온톨로지

현황: Workday Skills Cloud는 2018년에 출시된 머신러닝 기반 스킬 온톨로지입니다. 처음에는 2,500만 개의 스킬 인스턴스에서 시작하여 현재는 50억 개 이상의 스킬 데이터 포인트를 처리하고 있습니다. 핵심은 "20만 개 이상의 스킬"이 데이터베이스에 등록되어 있으며, 머신러닝을 통해 5만 5,000개의 검증된 스킬로 통합·정규화된다는 점입니다.

기술 아키텍처

• 그래프 기술: 스킬 간 관계성 매핑
• 머신러닝: 스킬 추론 및 자동 분류
• LLM 기반 서비스: 직종명에서 스킬 추론 (2024년 도입)
• OOV 대응: 신규·미지의 직종 타이틀에도 대응

스케일과 데이터

• 스킬 DB: 20만+ 스킬 (정규화 후 5.5만)
• 근로자 데이터: 4,500만 명 이상
• 고객 수: 2,100개사 이상
• Fortune 500: 30% 이상이 도입

LLM 기반 스킬 추론 (2024-2025년의 혁신)

Workday는 대규모 언어 모델(LLM)을 활용한 "Skill Inference" 서비스를 개발했습니다. 직종 타이틀에서 관련 스킬을 자동으로 추론하여 Workday Skill Ontology에 연결합니다. 기존 접근 방식으로는 대응이 어려웠던 "Out of Vocabulary(OOV)" 문제(신규·미지의 직종)를 해결합니다. 합성 데이터와 휴먼 어노테이터를 통한 평가 벤치마크를 구현하여 지속적인 모델 개선을 실현하고 있습니다.

관련 링크: Workday Skills Cloud | LLM Skill Inference 기술 블로그 | Skills Cloud 기반 해설

전망: Workday Illuminate(차세대 AI)와의 통합으로 스킬 이해와 인재 배치의 자동화가 더욱 진화할 예정입니다. HCM 에코시스템 내에서의 원활한 통합이 최대 강점입니다.

2. Eightfold AI: 16억 커리어 프로필의 딥러닝

현황: Eightfold는 16억 개 이상의 커리어 프로필과 160만 개 이상의 유니크 스킬[1]을 기반으로 하는 딥러닝 기반 탤런트 인텔리전스 플랫폼입니다. 과거 경험뿐만 아니라 "잠재력(포텐셜)"을 예측하는 AI 매칭이 특징이며, 채용부터 육성, 내부 이동까지 전체 탤런트 라이프사이클을 커버합니다.

딥러닝 아키텍처

• 뉴럴 네트워크: 10억+ 프로필에서 학습
• 스킬 온톨로지: 160만 유니크 스킬[1]
• 매칭 변수: 50개 이상의 변수로 스코어링
• 편향 완화: 공정성을 중시한 설계

도입 성과 (고객 사례)

• Bayer: 스크리닝 시간 90% 단축
• Vodafone: 채용 비용 50% 절감
• Cloudflare: 소싱 생산성 30% 향상
• Activision: 47%가 탤런트 재발견에서 채용

포텐셜 기반 매칭의 작동 원리

Eightfold의 독자성은 "스킬 인접성(Skill Adjacency)" 예측에 있습니다. 현재 보유하지 않은 스킬이라도 커리어 궤적에서 습득 가능성을 산출합니다. 예를 들어, Python 엔지니어가 데이터 사이언티스트로 전환할 잠재력을 유사 커리어 경로 분석에서 예측합니다. 이를 통해 기존 키워드 매칭으로는 찾을 수 없는 "숨겨진 인재"를 발굴할 수 있습니다.

관련 링크: Eightfold AI | Talent Intelligence Platform | 플랫폼 상세

전망: 기업 가치 20억 달러를 넘는 유니콘 기업으로서 정부 기관용 인증(FedRAMP)도 취득했습니다. 에이전트형 AI 도입으로 더욱 자동화가 기대됩니다.

3. Gloat Skills Foundation: Workforce Graph를 통한 실시간 적응

현황: Gloat는 2023년 4월에 Skills Foundation을 발표하며 AI 기반 스킬 인프라를 제공하고 있습니다. 독자적인 "Workforce Graph"는 수십억 개의 데이터 포인트에서 학습하며, 오픈 웹, 정부 데이터베이스, 파트너 데이터, 익명화된 사용자 데이터를 통합합니다. 시장 변화에 따라 동적으로 진화하는 스킬 온톨로지가 특징입니다.

Workforce Graph 기술

• 그래프 기반 AI: 일·스킬·역할의 관계를 딥러닝
• 데이터 소스: 오픈 웹, 정부 DB, 파트너
• 실시간 업데이트: 시장 동향에 따른 자동 진화
• HR 시스템 통합: 기존 데이터와의 조화

도입 성과 (고객 사례)

• Seagate: 내부 이동 성공률 50%, 퇴직 비용 2,000만 달러 절감
• Standard Chartered: 신규 직책의 60%를 내부 후보로 매칭
• 인게이지먼트: 직원 참여도 25% 향상
• 도입 기업: Schneider Electric, PepsiCo, Nestlé, HSBC

탤런트 마켓플레이스의 혁신

Gloat의 탤런트 마켓플레이스는 직원의 스킬·지향·목표에 기반하여 내부 기회(프로젝트, 멘토십, 커리어 전환, 임시 배치)를 매칭합니다. 정적인 직무 기술서 기반이 아닌 스킬 기반의 동적 매칭으로 "일이 사람을 찾는" 경험을 실현합니다. Work Orchestration Platform으로서 스킬·일·기술을 통합적으로 오케스트레이션합니다.

관련 링크: Gloat | Skills Foundation | Talent Marketplace

전망: 2024년 Work Tech Vendor Top 25에 선정되었습니다. 내부 모빌리티와 스킬 기반 조직으로의 전환을 지원하는 주요 플레이어로 성장 중입니다.

Workday Skills Cloud가 최적인 경우

• 이미 Workday HCM 도입 완료
• 단일 플랫폼에서의 일관성 중시
• 단계적인 스킬 전략으로 충분
• 추가 비용 최소화 필요

Eightfold AI가 최적인 경우

• 최첨단 AI 기술 필요
• 채용부터 퇴직까지 통합 뷰 필요
• 포텐셜 기반 채용 실현 목표

Gloat Skills Foundation이 최적인 경우

• 내부 모빌리티와 직원 인게이지먼트 우선
• 긱 워크·프로젝트 기반 기회 제공 중시
• 시장 변화에 연동된 스킬 온톨로지 필요
• 탤런트 마켓플레이스 구축 목표

활용 아이디어

이러한 플랫폼을 활용할 때의 주요 접근 방식을 소개합니다.

1. 스킬 갭 분석 파일럿 - 특정 부서(예: IT 부서)에서 스킬 갭을 가시화하고 리스킬링 계획의 근거 데이터 수집

2. 내부 모빌리티 촉진 - 탤런트 마켓플레이스를 도입하여 사내 프로젝트에 스킬 기반 매칭 실시

3. 채용 효율화 - 포텐셜 기반 매칭으로 기존 키워드 검색으로는 찾을 수 없는 후보자 발굴

핵심 요약

핵심 포인트

  • 시장 급성장: 스킬 인텔리전스 플랫폼 시장 2024년 $24억 → 2033년 $167억 (CAGR 21.6%)
  • Workday의 혁신: LLM 기반 스킬 추론으로 미지의 직종 타이틀에서도 관련 스킬 자동 추출 가능
  • Eightfold의 우위: 16억 프로필 딥러닝으로 "포텐셜"을 예측하는 유일한 플랫폼
  • Gloat의 차별화: 시장 연동형 스킬 온톨로지와 내부 모빌리티 특화로 직원 인게이지먼트 향상 실현
  • 도입 효과: 채용 비용 50% 절감, 스크리닝 시간 90% 단축 등 ROI가 명확히 실증됨

종합 평가: 긍정적 (시장 성장률 21.6% CAGR과 도입 기업의 명확한 ROI 실적으로 스킬 인텔리전스 플랫폼은 HR Tech 필수 인프라로 자리매김)

주요 용어 설명

스킬 온톨로지 - 스킬의 구성 요소와 스킬 간의 관계성을 정의한 지식 체계. 단순 분류(택소노미)보다 복잡한 관련성을 매핑
스킬 추론(Skill Inference) - 직종 타이틀이나 경력에서 관련 스킬을 AI가 자동으로 추측·추출하는 기술
탤런트 마켓플레이스 - 사내 인재와 기회(프로젝트, 포지션)를 스킬 기반으로 매칭하는 내부 플랫폼
포텐셜 기반 매칭 - 현재 스킬뿐 아니라 습득 가능성이나 성장 궤적을 예측하여 매칭하는 AI 기법
Workforce Graph - 일, 스킬, 역할의 관계성을 딥러닝으로 모델링한 Gloat 고유의 AI 아키텍처
