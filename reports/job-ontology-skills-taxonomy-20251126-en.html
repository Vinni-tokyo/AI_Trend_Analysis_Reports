Key Concepts

Skills Taxonomy

A hierarchical classification system for skills. A tree structure that organizes skills into categories, subcategories, and specific skills. Example: IT > Programming > Python

Skills Ontology

A system that defines skills as an entity model and maps relationships between them. Defines interdependencies, hierarchies, and lateral relationships between skills. Supports semantic understanding of skill similarities.

Job Ontology

A system that defines occupations and maps their relationships with skills. Provides 360-degree views including required skills for jobs, related occupations, and career paths.
