職業オントロジーとスキル分類体系：
グローバル標準と主要プラットフォーム比較

O*NET、ESCO、ISCO、Lightcast、LinkedIn Skills Graph、Workday Skills Cloud
レポート作成日：2025年11月26日 | 標準版レポート

主要トレンド Top 5

1. O*NET-SOC：米国標準職業分類システム

現状：O*NETは米国労働省が管理する職業情報データベースであり、2018年SOC（Standard Occupational Classification）をベースに構築されています。1,016の職業タイトルと55,000以上の職務を網羅し、各職業のスキル、知識、能力、タスクに関する詳細データを提供しています。

構造と規模

• 職業数：1,016タイトル（923がデータレベル職業）
• 職務数：55,000以上
• コード体系：8桁（SOC 6桁 + O*NET 2桁）
• 最新版：O*NET-SOC 2019（2020年11月更新）

データ要素

• Knowledge（知識）：33カテゴリ
• Skills（スキル）：35カテゴリ
• Abilities（能力）：52カテゴリ
• Work Activities（業務活動）：41カテゴリ

関連リンク： O*NET-SOC Taxonomy | O*NET Code Connector | OccO (Occupation Ontology)

展望：OccO（Occupation Ontology）プロジェクトが2023年12月にOBO Foundryで公式リリースされ、O*NETデータのオントロジー表現が進んでいます。

2. ESCO：欧州スキル・能力・資格・職業分類

現状：ESCOは欧州委員会が2010年から開発している多言語分類システムです。3,039の職業と13,939のスキルを28言語（EU全公用語+アイスランド語、ノルウェー語、ウクライナ語、アラビア語）で提供しています。

3つの柱

• Occupations（職業）：3,039職種
• Skills/Competences（スキル/能力）：13,939スキル
• Qualifications（資格）：国別資格との連携
• 最新版：ESCO v.1.2.0（2024年5月）

技術仕様

• オントロジー形式：OWL（Web Ontology Language）
• 外部連携：ISCO-08、ISCED-FoETとマッピング
• ダウンロード形式：SKOS-RDF、ODS、CSV
• API：無料公開

関連リンク： ESCO Portal | ESCO Download | ESCO Ontology

展望：ESCOは欧州労働市場の共通言語として機能し、求職者と求人のスキルベースマッチングを支援しています。

3. Lightcast Skills Taxonomy：リアルタイムスキルデータ

現状：Lightcast（旧EMSI + Burning Glass）は、世界最大の統合労働市場データプロバイダーです。毎日40,000以上のソースからリアルタイムデータを収集し、33,000以上のスキルと75,000以上の職種タイトルを提供しています。

スキル分類

• スキル数：33,000+（2週間ごとに更新）
• 3階層構造：カテゴリ → サブカテゴリ → スキル
• スキルタイプ：専門スキル、共通スキル、ソフトウェアスキル
• カテゴリ数：31（2025年3月時点）

データ基盤

• 求人データ：10億件以上
• 職種タイトル：75,000+
• データソース：40,000以上/日
• API：オープンアクセス可能

関連リンク： Lightcast Skills Taxonomy | Lightcast Titles Taxonomy | Skills API

展望：Josh Bersin氏は「Lightcastはグローバルスキルテック市場のスイス」と評価しており、中立的なデータプロバイダーとしての地位を確立しています。

4. LinkedIn Skills Graph：39,000スキルの関係性マップ

現状：LinkedInのSkills Graphは、39,000のスキル、8.75億人のユーザー、5,900万社の企業間の関係性を動的にマッピングしています。KGBertなどの機械学習モデルを活用し、スキル間の関係性を自動予測しています。

規模と構造

• スキル数：39,000
• ユーザー数：8.75億人
• 企業数：5,900万社
• 関係タイプ：Knowledge Lineages（親子関係等）

AI活用

• KGBert：スキル関係性の深層学習予測
• 自動スケーリング：39,000スキルを機械学習で管理
• Skills Match：求職者・求人マッチング
• Talent Insights：人材インテリジェンス

関連リンク： LinkedIn Skills Graph | Skills Taxonomy Engineering

展望：2015年以降、職種に必要なスキルセットは25%変化しており、2027年までに倍増すると予測されています。LinkedInはスキルベースの労働市場を推進しています。

5. 商用スキルインテリジェンスプラットフォーム

現状：スキルインテリジェンスプラットフォーム市場は2024年に24億ドル規模に達し、2033年までにCAGR 18-22%で成長し、116-167億ドルに達すると予測されています。

Workday Skills Cloud

• 導入企業：Fortune 500の30%以上
• 顧客数：2,100+社、15万ユーザー
• スキルDB：20万+スキル、4,500万労働者データ
• 特徴：機械学習+グラフ技術

Eightfold AI / Gloat

• Eightfold：16億キャリアプロファイル、160万スキル
• Gloat Skills Foundation：ベクトル空間でスキル意味理解
• Novartis事例：4年分を2ヶ月で達成
• 市場予測：2033年に116-167億ドル

関連リンク： Workday Skills Cloud | Eightfold AI | Gloat

展望：WEFによると2030年までに労働者のコアスキルの40%が変化するとされ、動的に進化するオントロジーへの需要が高まっています。

競争環境と市場ポジショニング

4層市場構造

職業オントロジー・スキル分類市場は、国際標準、国家標準、商用データプロバイダー、エンタープライズプラットフォームの4層で構成されています。

カテゴリ 主要プレイヤー スキル/職業数 特徴
国際標準 ISCO-08 (ILO) 436職業（4桁） グローバル統計比較の基盤
地域標準 ESCO (EU), O*NET (US) 3,039職業/1,016職業 多言語対応、詳細スキルデータ
商用データ Lightcast, LinkedIn 33,000+/39,000スキル リアルタイム更新、AI活用
エンタープライズ Workday, Eightfold, Gloat 20万+スキル、4,500万+労働者 タレントマーケットプレイス統合

主要プラットフォーム比較チャート

市場動向（5つのキーポイント）

  • 市場規模急成長：スキルインテリジェンスプラットフォーム市場は2024年24億ドル→2033年116-167億ドル（CAGR 18-22%）
  • スキル寿命の短縮：25年前はスキルが26年間有効だったが、現在は5年未満
  • スキルギャップの深刻化：87%の企業がスキルギャップを実感、HR責任者の60%が最優先課題と回答
  • AI駆動オントロジー：静的タクソノミーから動的・自己進化型オントロジーへ移行
  • アジア太平洋急成長：APAC市場は2024年4.8億ドル、CAGR 21.5%で最速成長

活用アイデア

市場で観察される主要な活用アプローチをご紹介します。

1. ESCO/O*NETマッピング - 自社の職務記述書をESCOまたはO*NETにマッピングし、標準化されたスキル言語で人材管理を改善

2. Lightcast API統合 - リアルタイム労働市場データを活用し、採用戦略やスキルギャップ分析を高度化

3. スキルオントロジー構築 - Gloat/Workday等のプラットフォームを導入し、社内タレントマーケットプレイスを実現

エグゼクティブサマリー

重要なポイント

  • グローバル標準の並立：ISCO（国際）、ESCO（欧州）、O*NET（米国）が地域標準として機能
  • 商用データの台頭：Lightcast 33,000+スキル、LinkedIn 39,000スキルがリアルタイム更新
  • 市場急成長：スキルインテリジェンス市場は2024年24億ドル→2033年116-167億ドル
  • AI駆動進化：静的タクソノミーから動的オントロジーへ、KGBertなどの機械学習活用
  • スキルベース経済：2027年までに職種スキルセットの50%が変化、スキルファースト採用が加速

導入ガイド

国際比較や統計が必要な場合 → ISCO-08 / ESCO
理由：ILO/EU標準として多言語対応、無料利用可能

米国市場や詳細職業データが必要な場合 → O*NET
理由：1,016職業の詳細スキル・能力データ、無料公開

リアルタイム労働市場分析が必要な場合 → Lightcast
理由：33,000スキル、2週間更新、API利用可能

エンタープライズ人材管理が必要な場合 → Workday / Eightfold / Gloat
理由：タレントマーケットプレイス統合、AI駆動マッチング

総合評価：ポジティブ（スキルベース経済への移行加速、AI駆動オントロジーの進化、市場急成長により、職業・スキル分類体系は人材戦略の中核インフラに）

主要用語の説明

スキルタクソノミー（Skills Taxonomy） - スキルを階層的に分類・整理するフレームワーク。グループとサブグループに基づく構造化アプローチ
スキルオントロジー（Skills Ontology） - スキル間の関係性と相互依存性をマッピングする高度なモデル。タクソノミーより複雑で動的
O*NET-SOC - 米国労働省の職業情報ネットワーク。1,016職業の詳細データを提供する米国標準
ESCO - European Skills, Competences, Qualifications and Occupations。欧州委員会が開発した28言語対応の職業・スキル分類
タレントマーケットプレイス - 社内の人材とプロジェクト/ポジションをスキルベースでマッチングするプラットフォーム（例：Workday、Gloat）
