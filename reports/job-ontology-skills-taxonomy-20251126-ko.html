글로벌 표준과 주요 플랫폼 비교

O*NET, ESCO, ISCO, Lightcast, LinkedIn Skills Graph, Workday Skills Cloud
리포트 작성일: 2025년 11월 26일 | 표준 리포트

주요 트렌드 Top 5

1. O*NET-SOC: 미국 표준 직업 분류 시스템

현황: O*NET은 미국 노동부가 관리하는 직업 정보 데이터베이스로, 2018년 SOC(Standard Occupational Classification)를 기반으로 구축되어 있습니다. 1,016개의 직업 타이틀과 55,000개 이상의 직무를 포괄하며, 각 직업의 스킬, 지식, 능력, 태스크에 관한 상세 데이터를 제공합니다.

구조와 규모

• 직업 수: 1,016개 타이틀 (923개가 데이터 레벨 직업)
• 직무 수: 55,000개 이상
• 코드 체계: 8자리 (SOC 6자리 + O*NET 2자리)
• 최신 버전: O*NET-SOC 2019 (2020년 11월 업데이트)

데이터 요소

• Knowledge(지식): 33개 카테고리
• Skills(스킬): 35개 카테고리
• Abilities(능력): 52개 카테고리
• Work Activities(업무 활동): 41개 카테고리

관련 링크: O*NET-SOC Taxonomy | O*NET Code Connector | OccO (Occupation Ontology)

전망: OccO(Occupation Ontology) 프로젝트가 2023년 12월 OBO Foundry에서 공식 릴리스되어, O*NET 데이터의 온톨로지 표현이 진행되고 있습니다.

2. ESCO: 유럽 스킬·역량·자격·직업 분류

현황: ESCO는 유럽 위원회가 2010년부터 개발해 온 다국어 분류 시스템입니다. 3,039개의 직업과 13,939개의 스킬을 28개 언어(EU 전체 공용어 + 아이슬란드어, 노르웨이어, 우크라이나어, 아랍어)로 제공합니다.

3대 축

• Occupations(직업): 3,039개 직종
• Skills/Competences(스킬/역량): 13,939개 스킬
• Qualifications(자격): 국가별 자격과 연계
• 최신 버전: ESCO v.1.2.0 (2024년 5월)

기술 사양

• 온톨로지 형식: OWL (Web Ontology Language)
• 외부 연계: ISCO-08, ISCED-FoET과 매핑
• 다운로드 형식: SKOS-RDF, ODS, CSV
• API: 무료 공개

관련 링크: ESCO Portal | ESCO Download | ESCO Ontology

전망: ESCO는 유럽 노동 시장의 공통 언어로 기능하며, 구직자와 구인의 스킬 기반 매칭을 지원하고 있습니다.

3. Lightcast Skills Taxonomy: 실시간 스킬 데이터

현황: Lightcast(구 EMSI + Burning Glass)는 세계 최대 통합 노동 시장 데이터 제공 기업입니다. 매일 40,000개 이상의 소스에서 실시간 데이터를 수집하며, 33,000개 이상의 스킬과 75,000개 이상의 직종 타이틀을 제공합니다.

스킬 분류

• 스킬 수: 33,000+ (2주마다 업데이트)
• 3계층 구조: 카테고리 → 서브카테고리 → 스킬
• 스킬 타입: 전문 스킬, 공통 스킬, 소프트웨어 스킬
• 카테고리 수: 31개 (2025년 3월 기준)

데이터 기반

• 채용 공고 데이터: 10억 건 이상
• 직종 타이틀: 75,000+
• 데이터 소스: 40,000개 이상/일
• API: 오픈 액세스 가능

관련 링크: Lightcast Skills Taxonomy | Lightcast Titles Taxonomy | Skills API

전망: Josh Bersin은 "Lightcast는 글로벌 스킬테크 시장의 스위스"라고 평가하며, 중립적 데이터 제공 기업으로서의 위치를 확립하고 있습니다.

4. LinkedIn Skills Graph: 39,000 스킬의 관계성 맵

현황: LinkedIn의 Skills Graph는 39,000개 스킬, 8.75억 명의 사용자, 5,900만 개 기업 간의 관계를 동적으로 매핑하고 있습니다. KGBert 등 머신러닝 모델을 활용하여 스킬 간 관계를 자동 예측합니다.

규모와 구조

• 스킬 수: 39,000개
• 사용자 수: 8.75억 명
• 기업 수: 5,900만 개
• 관계 타입: Knowledge Lineages (부모-자식 관계 등)

AI 활용

• KGBert: 스킬 관계의 딥러닝 예측
• 자동 스케일링: 39,000 스킬을 머신러닝으로 관리
• Skills Match: 구직자·구인 매칭
• Talent Insights: 인재 인텔리전스

관련 링크: LinkedIn Skills Graph | Skills Taxonomy Engineering

전망: 2015년 이후 직종에 필요한 스킬셋은 25% 변화했으며, 2027년까지 2배로 증가할 것으로 예측됩니다. LinkedIn은 스킬 기반 노동 시장을 선도하고 있습니다.

5. 상용 스킬 인텔리전스 플랫폼

현황: 스킬 인텔리전스 플랫폼 시장은 2024년 24억 달러 규모에 도달했으며, 2033년까지 CAGR 18-22%로 성장하여 116-167억 달러에 이를 것으로 예측됩니다.

Workday Skills Cloud

• 도입 기업: Fortune 500의 30% 이상
• 고객 수: 2,100개 이상, 15만 사용자
• 스킬 DB: 20만+ 스킬, 4,500만 근로자 데이터
• 특징: 머신러닝 + 그래프 기술

Eightfold AI / Gloat

• Eightfold: 16억 커리어 프로필, 160만 스킬
• Gloat Skills Foundation: 벡터 공간에서 스킬 의미 이해
• Novartis 사례: 4년치를 2개월에 달성
• 시장 예측: 2033년 116-167억 달러

관련 링크: Workday Skills Cloud | Eightfold AI | Gloat

전망: WEF에 따르면 2030년까지 노동자 핵심 스킬의 40%가 변화할 것으로 예측되며, 동적으로 진화하는 온톨로지에 대한 수요가 증가하고 있습니다.

경쟁 환경과 시장 포지셔닝

4계층 시장 구조

직업 온톨로지·스킬 분류 시장은 국제 표준, 국가 표준, 상용 데이터 제공사, 엔터프라이즈 플랫폼의 4개 계층으로 구성되어 있습니다.

카테고리 주요 플레이어 스킬/직업 수 특징
국제 표준 ISCO-08 (ILO) 436개 직업 (4자리) 글로벌 통계 비교의 기반
지역 표준 ESCO (EU), O*NET (US) 3,039개 직업/1,016개 직업 다국어 지원, 상세 스킬 데이터
상용 데이터 Lightcast, LinkedIn 33,000+/39,000개 스킬 실시간 업데이트, AI 활용
엔터프라이즈 Workday, Eightfold, Gloat 20만+ 스킬, 4,500만+ 근로자 탤런트 마켓플레이스 통합

주요 플랫폼 비교 차트

시장 동향 (5가지 핵심 포인트)

  • 시장 규모 급성장: 스킬 인텔리전스 플랫폼 시장 2024년 24억 달러 → 2033년 116-167억 달러 (CAGR 18-22%)
  • 스킬 수명 단축: 25년 전 스킬은 26년간 유효했으나, 현재는 5년 미만
  • 스킬 갭 심화: 87% 기업이 스킬 갭 실감, HR 책임자 60%가 최우선 과제로 응답
  • AI 기반 온톨로지: 정적 분류 체계에서 동적·자기 진화형 온톨로지로 전환
  • 아시아태평양 급성장: APAC 시장 2024년 4.8억 달러, CAGR 21.5%로 최고 성장

활용 아이디어

시장에서 관찰되는 주요 활용 접근법을 소개합니다.

1. ESCO/O*NET 매핑 - 자사 직무 기술서를 ESCO 또는 O*NET에 매핑하여 표준화된 스킬 언어로 인재 관리 개선

2. Lightcast API 통합 - 실시간 노동 시장 데이터를 활용하여 채용 전략과 스킬 갭 분석 고도화

3. 스킬 온톨로지 구축 - Gloat/Workday 등 플랫폼을 도입하여 사내 탤런트 마켓플레이스 실현

경영진 요약

핵심 포인트

  • 글로벌 표준 병립: ISCO(국제), ESCO(유럽), O*NET(미국)이 지역 표준으로 기능
  • 상용 데이터 부상: Lightcast 33,000+ 스킬, LinkedIn 39,000 스킬이 실시간 업데이트
  • 시장 급성장: 스킬 인텔리전스 시장 2024년 24억 달러 → 2033년 116-167억 달러
  • AI 기반 진화: 정적 분류 체계에서 동적 온톨로지로, KGBert 등 머신러닝 활용
  • 스킬 기반 경제: 2027년까지 직종 스킬셋 50% 변화, 스킬 우선 채용 가속

도입 가이드

국제 비교나 통계가 필요한 경우 → ISCO-08 / ESCO
이유: ILO/EU 표준으로 다국어 지원, 무료 이용 가능

미국 시장이나 상세 직업 데이터가 필요한 경우 → O*NET
이유: 1,016개 직업의 상세 스킬·능력 데이터, 무료 공개

실시간 노동 시장 분석이 필요한 경우 → Lightcast
이유: 33,000 스킬, 2주 업데이트, API 이용 가능

엔터프라이즈 인재 관리가 필요한 경우 → Workday / Eightfold / Gloat
이유: 탤런트 마켓플레이스 통합, AI 기반 매칭

종합 평가: 긍정적 (스킬 기반 경제로의 전환 가속, AI 기반 온톨로지 진화, 시장 급성장으로 직업·스킬 분류 체계는 인재 전략의 핵심 인프라로 자리매김)

주요 용어 설명

스킬 택소노미(Skills Taxonomy) - 스킬을 계층적으로 분류·정리하는 프레임워크. 그룹과 서브그룹에 기반한 구조화 접근법
스킬 온톨로지(Skills Ontology) - 스킬 간 관계와 상호 의존성을 매핑하는 고급 모델. 택소노미보다 복잡하고 동적
O*NET-SOC - 미국 노동부의 직업 정보 네트워크. 1,016개 직업의 상세 데이터를 제공하는 미국 표준
ESCO - European Skills, Competences, Qualifications and Occupations. 유럽 위원회가 개발한 28개 언어 대응 직업·스킬 분류
탤런트 마켓플레이스 - 사내 인재와 프로젝트/포지션을 스킬 기반으로 매칭하는 플랫폼 (예: Workday, Gloat)
