기술 개관

추천 시스템의 딥러닝 알고리즘은 급속히 진화하고 있으며, 기존의 협업 필터링을 넘어선 새로운 세대의 기술들이 부상하고 있습니다. Transformer, 대조 학습, 지식 그래프, 생성 모델 등 다양한 접근법이 연구·구현되어 정확도 향상과 콜드스타트 문제 해결에 기여하고 있습니다.

Transformer Self-Attention Contrastive Learning Knowledge Graph Multimodal Fusion VAE Diffusion Model GNN

주목 기술 Top 5

1. Transformer 기반 순차 추천 모델 (SASRec·BERT4Rec·HydraRec)

개요: Self-Attention 메커니즘을 활용한 순차 추천 모델은 사용자의 행동 이력에서 동적인 선호도를 포착하는 최신 기법으로 자리잡았습니다. SASRec과 BERT4Rec이 양대 베이스라인으로 널리 사용되고 있습니다.

대표 모델

• SASRec: 좌→우 단방향 모델, 네거티브 샘플링으로 BCE 손실
• BERT4Rec: 양방향 모델, Cloze 태스크로 학습, 전체 아이템 소프트맥스
• HydraRec (2025): 효율적 Attention, 시퀀스 길이에 대한 이차 계산량 개선

최신 연구 (2025)

• RSS (Recency Sampling): SASRec에 RSS 적용 시 NDCG 60% 향상, BERT4Rec 대비 16% 향상, 학습 시간 93% 단축
• eSASRec (RecSys 2025): 모듈러 방식으로 확장성 향상
• TIGER: SASRec, BERT4Rec, S3-Rec, P5를 모든 데이터셋에서 능가

관련 링크: HydraRec Paper | BERT4Rec Paper | NVIDIA Transformers4Rec

구현 포인트: 동일 손실 함수 사용 시 SASRec이 BERT4Rec보다 품질·학습 속도 모두 우수한 경향. NVIDIA의 Transformers4Rec 라이브러리가 Hugging Face와 통합되어 구현이 용이해짐.

2. 대조 학습(Contrastive Learning) 기반 자기 지도 추천

개요: SimCLR에서 영감을 받은 대조 학습 프레임워크가 추천 시스템에 적용되어, 데이터 증강과 페어 학습을 통해 로버스트성과 정확도가 크게 향상되고 있습니다.

주요 접근법

• SGL (Self-supervised Graph Learning): SimCLR 패러다임을 사용자·아이템 그래프에 적용
• SCL (Supervised Contrastive Learning): 유사 인터랙션 이력을 가진 사용자의 유사성 학습
• FD-DCL (2025): 주파수 도메인 분리와 이중 대조 학습 결합

기술적 특징

• InfoNCE 손실: 양성 페어 유사도 최대화, 음성 페어 유사도 최소화
• 데이터 증강: 그래프 드롭아웃, 노드 마스킹, 엣지 섭동
• 표현 학습: 다른 증강 뷰 간 엔티티 표현 정렬

관련 링크: SimCLR GitHub | SCL for Recommendation | SimCLR Original Paper

산업 적용: Spotify(세션 기반 추천), Pinterest(콘텐츠 추천) 등에서 실운용. LightGCN과의 조합으로 Gowalla, Yelp2018, Amazon-Book 데이터셋에서 유효성 입증.

3. 지식 그래프 임베딩(KGE)과 이종 정보 네트워크

개요: 지식 그래프는 풍부한 멀티 관계 정보를 포함하며, 아이템 간 보완성과 사용자의 잠재적 관심사를 포착하는 데 유효합니다. 이종 정보 네트워크(HIN)로서의 모델링이 주류가 되고 있습니다.

주요 기법

• TransE/TransR: 관계를 벡터 공간에서의 변환으로 학습 (CFKG, CKE에서 사용)
• HKGAT (2025): 이종 지식 그래프 Attention 네트워크, 설명 가능한 추천
• 메타패스: 사용자·아이템 간 유사성을 다차원으로 특성화

최신 동향 (2025)

• 계층적 그래프 Attention: 멀티모달 지식 그래프와의 통합
• GCL+KGE: 그래프 대조 학습과 지식 그래프 임베딩의 융합
• KHGCN: 계층적 그래프 캡슐 네트워크

관련 링크: HKGAT (2025) | Multi-level Contrastive KG | Iterative HG Learning

과제와 해결책: 이종 엔티티의 균일 임베딩 차원 설정이 어려움. 선형 메타패스로는 복잡한 관계 계층을 표현하기 어려워 계층적 Attention 메커니즘으로 대응.

4. 멀티모달 추천 시스템 (CLIP 융합·MetaMMF)

개요: 이미지, 텍스트, 오디오 등 여러 모달리티를 통합한 추천 시스템이 급속히 발전하고 있습니다. CLIP 등의 대규모 사전 학습 모델을 활용하여 의미적으로 정렬된 임베딩을 생성합니다.

기술 아키텍처

• 특징 추출: CNN/ViT(이미지), BERT/SentenceTransformer(텍스트), GNN(그래프)
• CLIP 통합: 시각·언어 의미 정렬, 단 내적 한계 존재
• 융합 전략: 조기 융합(저수준), 후기 융합(예측 수준), 메타 학습

최신 연구 (2025)

• MetaMMF: 메타 학습으로 마이크로비디오별 동적 파라미터 할당
• CLIP 가이드 특징 정렬: Self-Attention과 Cross-Attention으로 모달리티 간 상관 강화
• 차분 프라이버시: 프라이버시 보호 멀티모달 추천

관련 링크: MetaMMF Paper | Multimodal Alignment Survey | Visual-Textual Fusion

신흥 트렌드: LLM, 에이전트형 계획, 지속적 메모리, 적응형 행동 모듈을 활용한 멀티모달·컨텍스트 인식·자율형 추천이 중요한 연구 방향으로 부상.

5. 생성형 추천 모델 (VAE·확산 모델·RAG)

개요: 변분 오토인코더(VAE)와 확산 모델을 활용한 생성형 추천은 데이터 희소성과 콜드스타트 문제에 강하며, 베이지안 추론을 효율적으로 수행할 수 있습니다.

VAE 기반 추천

• 확률적 프레임워크: 잠재 공간상의 확률 분포 학습으로 데이터 희소성에 강함
• 다기준 추천: 한 기준의 선호가 다른 기준에 영향을 미치는 관계 발견
• 과제: 노출 편향, 사후 붕괴(posterior collapse)

확산 모델 & RAG

• CODIGEM: 확산 프로세스를 추천에 적용한 최초의 생성 모델
• RAG 통합: 벡터 DB 시장 17.3억 달러→106억 달러 (2032년), 서브 100ms 레이턴시
• Pinecone/Weaviate: RAG 추천용 벡터 DB

관련 링크: VAE in RecSys Survey | Pinecone RAG Guide | VAE Multi-Criteria RecSys

구현 동향: 시맨틱 ID를 사용한 생성적 검색 추천 모델의 PyTorch 구현이 공개됨. 잠재 확산 모델에서 VAE와 조건부 U-Net을 결합한 최신 기법도 등장.

2025년 기술 동향 포인트

  • 효율성 중시: HydraRec 등 계산 효율을 개선한 새로운 아키텍처 부상
  • 하이브리드 접근: 대조 학습+KG, Transformer+GNN 등 조합 기법이 주류

  • 설명 가능성: HKGAT 등 2025년 연구에서 설명 가능한 추천에 대한 관심 고조
  • 프라이버시 보호: 차분 프라이버시 적용 멀티모달 추천 연구 진전

활용 아이디어

각 기술의 실질적인 적용 접근법을 소개합니다.

1. Transformers4Rec 도입 - NVIDIA 라이브러리로 SASRec/BERT4Rec 신속 프로토타이핑, Hugging Face 통합으로 구현 용이

2. 대조 학습+LightGCN - 기존 GCN 기반 추천에 SGL/SCL 추가하여 로버스트성 향상 검증

3. RAG 추천 파일럿 - Pinecone/Weaviate로 상품 벡터 DB 구축, LLM과 결합한 대화형 추천 실험

핵심 요약

주요 포인트

  • Transformer형이 주류: SASRec/BERT4Rec이 양대 베이스라인, RSS 적용으로 NDCG 60% 향상과 학습 시간 93% 단축 동시 달성
  • 대조 학습의 보급: SimCLR 패러다임이 SGL/SCL로서 추천에 침투, Spotify·Pinterest에서 실운용
  • 지식 그래프×설명 가능성: HKGAT 등 2025년 연구에서 설명 가능한 추천 대응 진전
  • 멀티모달 융합: CLIP 통합과 메타 학습(MetaMMF)으로 시각·텍스트·오디오 통합 정확도 향상
  • RAG 추천의 부상: 벡터 DB 시장 10배 성장(17억→106억 달러), 서브 100ms 레이턴시로 실용화

주요 용어 설명

Self-Attention - 입력 시퀀스 내 각 요소가 다른 모든 요소와의 관련도를 학습하는 메커니즘. Transformer의 핵심 기술
대조 학습 (Contrastive Learning) - 유사 페어를 가깝게, 비유사 페어를 멀리하는 자기 지도 학습. SimCLR이 대표 예시
지식 그래프 임베딩 (KGE) - 엔티티와 관계를 벡터 공간에 임베딩하여 추론에 활용하는 기술
CLIP - OpenAI의 시각·언어 사전 학습 모델. 이미지와 텍스트의 의미적 정렬을 학습
VAE (변분 오토인코더) - 확률적 잠재 변수 모델. 입력을 확률 분포로 인코딩하고 샘플링으로 생성
RAG (검색 증강 생성) - 외부 지식 베이스에서 관련 정보를 검색하여 생성 모델의 입력에 추가하는 기법
