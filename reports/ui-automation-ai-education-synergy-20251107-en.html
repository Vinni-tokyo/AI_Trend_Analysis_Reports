Key Terminology

Vision-Based UI Agent - Technology enabling AI to "see" screenshots and recognize buttons/menus for human-like operation. OmniParser V2 is a prime example.
Self-Healing Test - Functionality where AI automatically adjusts test code to maintain operation even when UI changes, dramatically reducing maintenance burden.
Cross-Platform Testing - Running the same tests across smartphones, PCs, tablets, and other devices to verify functionality. Essential for educational apps.
WCAG (Web Content Accessibility Guidelines) - International standard for web accessibility, defining criteria for UIs usable by visually and hearing impaired individuals.
Flaky Test - Unstable tests that succeed or fail inconsistently with the same code. Playwright's auto-wait functionality significantly reduces these.
Visual Regression Testing - Method of comparing screenshots before and after UI changes to detect unintended design modifications. Applitools excels at this.
Natural Language Testing - Technology enabling test writing in human language like "click login button" without programming. testRigor is a representative example.
